Pengumuman

Terima kasih telah mencoba program yang sudah saya buat. Dalam program CRUD ini terdapat beberapa bug atau masalah yang belum dapat saya selesaikan dengan logika pemrograman saya menggunakan javascript. seperti :

  1. waktu menghapus (delete) bisa menghapus akan tetapi yang dihapus adalah baris yang terakhir, bukan baris yang diklik.
  2. waktu edit data, bisa memunculkan pop up edit lalu bisa menambahkan data yang telah di edit, namun bug atau masalah nya adalah tidak bisa menghapus data yang telah ter edit tadi, jadi sama saja dengan waktu tambah/create.

Demikian informasi yang dapat saya sampaikan, mungkin untuk revisi atau cara untuk menangani bug program CRUD atau ada beberapa saran lain mengenai saya dapat dikirim melalui email rikitoro22@gmail.com atau lewat sosmed dibawah!. Terima kasih.
